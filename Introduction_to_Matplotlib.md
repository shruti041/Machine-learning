# What is Matplotlib?
![Matplotlib](https://360digit.b-cdn.net/uploads/blog/e417d8bc9e498cee7d9922c8c4342915.png)
Matplotlib is a powerful and versatile open-source plotting library for Python, designed to help users visualize data in a variety of formats. If you want to convert your boring data into interactive plots and graphs, Matplotlib is the tool for you.
## Why matplotlib?
- Built on numpy arrays (and python)
- Integrates directly with pandas

- Simple to use interface (once you get the foundations)
